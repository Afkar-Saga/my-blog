My First Ever JavaScript Game

Posted on Monday, December 9th 2019

After some time of studying JavaScript, I'm excited to make a game. Because I'm not really a pro at JavaScript yet, I'm trying to make a simple game. It took a long time to think what kind of game that I want to make. But then I remember my favorite game, which is Pokemon, so I start to make a simple dodge game, but the character is Pokemon. I called it, "Pokemon Fly". After some trial and error, I finally made it, but, it's only one character, which is Mudkip, my favorite Pokemon. You can check it if you want. Click here to play my game. It's still a work in progress. The next project is to create Level 2, with Charmander. So, I guess that's it for this post. Byeee..!!

Pokemon Fly 1.1

Posted on Saturday, September 3rd 2022

It's been 3 years since I made Pokemon Fly, and now I finally come back to it and made an update, "Pokemon Fly 1.1". I added a new Pokemon, Charmander like I said I would. I made the new level quite difficult, even I'm having a hard time trying to beat it. I fixed some bugs, and also added a new feature, the Level button. It lets you switch between levels after you've unlocked it. In the future I will make another level so that this feature is more useful. That's it for now for this post, I hope everyone can enjoy my game. Cya!
